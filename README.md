# 🛒 SalesSavvy - Full Stack E-Commerce Platform

SalesSavvy is a full-stack e-commerce web application developed using React.js, Spring Boot, and MySQL. The platform enables users to browse products, manage carts, place orders, and securely access their accounts through authentication and authorization features.

## 🚀 Features

### User Features

* User Registration & Login
* JWT Authentication
* Product Browsing
* Product Search
* Add to Cart
* Remove from Cart
* Place Orders
* Order History
* Responsive User Interface

### Admin Features

* Product Management
* Add New Products
* Update Product Details
* Delete Products
* Manage Inventory

## 🛠️ Tech Stack

## ⚙️ Installation & Setup

### Clone Repository

git clone https://github.com/sandesh0303/SalesSavvy.git

### Backend Setup

cd backend

mvn spring-boot:run

### Frontend Setup

cd frontend

npm install

npm run dev
